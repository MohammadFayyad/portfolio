# Mohamed Fayyad — Flutter Mobile Developer Portfolio

A premium, production-ready personal portfolio showcasing Flutter mobile development expertise, projects, and skills. Built with Next.js 16, TypeScript, Tailwind CSS v4, Framer Motion, and a reusable modular component architecture.

## Features

- ✨ Beautiful dark/light theme toggle with next-themes
- 🎨 Premium UI with Tailwind CSS v4 and custom components
- ⚡ Server-side rendering and static generation with Next.js App Router
- 📱 Fully responsive design
- 🔍 SEO optimized with metadata API, robots.txt, and sitemap
- 📊 Contact form with react-hook-form and Zod validation
- 🎯 Type-safe with TypeScript
- 🚀 Production-ready and deployment-ready

## Scripts

- `npm run dev` – start the development server on http://localhost:3000
- `npm run build` – build the production app
- `npm run start` – start the production server
- `npm run lint` – run ESLint

## Content Sections

- **Home**: Hero with current focus and key highlights
- **About**: Background, education, and expertise
- **Skills**: Organized skill groups (Languages, Mobile, Architecture, APIs, Storage, Tools)
- **Projects**: 3 featured Flutter & React projects with descriptions and tech stacks
- **Experience**: Work experience and professional history
- **Resume**: Education and key highlights
- **Contact**: Contact form and direct messaging

## Tech Stack

- Next.js 16.2.10 (App Router)
- React 19.2.4
- TypeScript 5
- Tailwind CSS v4
- Framer Motion 12
- next-themes (dark mode)
- react-hook-form + Zod (form validation)
- Lucide React (icons)
